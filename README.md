# KaoWeather


<h1>이모티콘 날씨 알림 웹 애플리케이션</h1>
[React.js 기반]
<br>
무료 날씨 API OpenWeather의 API를 사용했습니다. (https://openweathermap.org/api)
<br>
얼굴(顔, かお) 이모티콘과 날씨 알림이를 접목시킨 웹기반 앱플리케이션,
